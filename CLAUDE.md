## Project Overview

This is the **Mass. Numbers** blog (www.massnumbers.us) — a data-driven blog about Massachusetts politics and policy. It is built with [Distill for R Markdown](https://rstudio.github.io/distill/blog.html) and hosted on GitHub Pages.

## Architecture

- **`_site.yml`** — Distill site configuration (title, navbar, output directory, theme, Google Analytics)
- **`_posts/`** — Blog post source directories, each named `YYYY-MM-DD-slug/` containing an `.Rmd` file and any post-specific CSV data files
- **`docs/`** — Compiled HTML output (the published site). Generated by knitting; do not edit directly
- **`_data/`** — Shared data files (CSV, XLSX) organized by year subdirectories (`2020/`, `2023/`, `2024/`)
- **`_gis/`** — GIS shapefiles and GeoJSON for Massachusetts districts, towns, precincts, and neighborhoods
- **`images/`** — Site images and post preview images
- **`massnumbers.css`** — Custom Distill theme (Amiri font, blue header)
- **`CNAME`** — Custom domain: www.massnumbers.us

## Building the Site

Requires R with the `distill` package. From R or RStudio:

```r
# Render the entire site
rmarkdown::render_site()

# Render a single post
rmarkdown::render("_posts/YYYY-MM-DD-slug/slug.Rmd")
```

Output goes to the `docs/` directory which is served by GitHub Pages.

## Blog Post Structure

Each post `.Rmd` follows this pattern:
1. YAML front matter with `distill::distill_article` output, `self_contained: false`
2. Setup chunk loading R libraries (code output hidden with `echo = FALSE`)
3. `meta()` block for social media metadata (Open Graph, Twitter cards) via `metathis`
4. Data loading from local CSVs or `_data/` directory
5. Prose interspersed with R code chunks producing tables and maps

## Commonly Used R Packages

- **tidyverse** — data wrangling (used in every post)
- **sf** / **tmap** / **tmaptools** — spatial data and interactive maps (Leaflet-based)
- **gt** / **gtExtras** — formatted tables
- **plotly** — interactive charts
- **formattable** — formatted numeric display
- **metathis** — social media meta tags
- **rstanarm** — Bayesian regression (used in some analytical posts)

## Content Domain

Posts analyze Massachusetts election data: precinct-level results, PVI (Partisan Voter Index) ratings, census demographics, state legislative district maps, campaign finance (OCPF) data, and special election previews. Data files use MA-specific terminology (VTDs, precincts, state rep/senate districts, congressional districts).
